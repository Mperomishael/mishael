SETUP REQUIRED
================================================================================

User Must Do:

1. Create Resend Account
   - Go to: https://resend.com
   - Sign up with email
   - Free tier included

2. Get Resend API Key
   - From Resend dashboard → API Keys
   - Create new key
   - Copy the key

3. Create .env.local
   - Location: /app/.env.local
   - Content: VITE_RESEND_API_KEY=your_key_here
   - Keep private, don't commit to git

4. Create Firestore Collection
   - Firebase Console → Firestore Database
   - Create collection: "subscribers"
   - Auto-creates documents on first subscription

5. Install & Run
   - cd app
   - npm install (installs Resend)
   - npm run dev

HOW IT WORKS
================================================================================

User Flow - Subscription:
  1. User finds footer on any page
  2. Enters name and email
  3. Clicks Subscribe
  4. Welcome email sent automatically
  5. Subscriber appears in admin dashboard
  6. User gets "Thank you" message

Admin Flow - Newsletter:
  1. Login to /admin/dashboard
  2. Click "Email Marketing" tab
  3. Click "Newsletter" sub-tab
  4. Write subject and content
  5. Click "Send to X Subscribers"
  6. All subscribers receive email
  7. Last email sent timestamp updates

Admin Flow - Project Announcement:
  1. Add new project to portfolio
  2. Go to admin dashboard
  3. Click "Email Marketing" tab
  4. Click "Announcements" sub-tab
  5. Enter project name and category
  6. Click "Send Announcement"
  7. Auto-formatted email sent to all subscribers

  ├─ email: string (unique)

  ├─ subscribedAt: timestamp
  ├─ status: "active" | "unsubscribed"

  └─ lastEmailSent: timestamp
